## 개요

이 문서는 불일치 탐지 테스트를 위한 샘플입니다.

---

## 교차 검증 위반 사례

### 1. 삭제된 파일 참조

자세한 내용은 `src/deleted-module.ts`를 참고하세요. (파일이 실제로 존재하지 않음)

### 2. 존재하지 않는 함수 참조

`nonExistentFunction()`을 사용하여 데이터를 처리합니다. (함수가 코드베이스에 없음)

### 3. 중복 정보

이 섹션의 내용은 다른 문서와 80% 이상 겹칩니다.
(실제 중복 탐지는 2개 파일 비교 시에만 발생)

## 추적가능성 위반 사례

### 1. 파일:라인 번호 누락

`calculateTotal` 함수를 사용하여 합계를 계산합니다. (파일:라인 번호 없음)

### 2. API 시그니처 불일치

**calculateTotal()**의 시그니처는 `(items: string[]) => string`입니다.
(실제 코드는 `(items: number[]) => number`이므로 불일치)
