plugins/hooks/tests/KEYWORD-IMPROVEMENT-SUGGESTIONS.md 를 기반으로
plugins/cache/skill-metadata.json 의 keywords를 업데이트해줘.

규칙:

1. 기존 키워드는 유지
2. 제안된 키워드 중 명확한 것만 추가
3. 너무 일반적인 키워드(예: "해줘", "좀")는 제외
4. 다른 스킬과 충돌하는 키워드는 주의

업데이트 후 다시 테스트를 실행해서 개선 여부 확인:

- 이전 성공률: N%
- 이후 성공률: N%
- 개선된 케이스: [ID 목록]
- 악화된 케이스: [ID 목록]
