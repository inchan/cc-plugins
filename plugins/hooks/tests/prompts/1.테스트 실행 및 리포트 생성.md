plugins/hooks/tests/fixtures/test-prompts.json 파일을 읽고
plugins/hooks/skill-recommend-hook.py 를 각 테스트 케이스에 대해 실행해줘.

실행 방법:
echo '{"prompt": "<테스트 프롬프트>"}' | python3 plugins/hooks/skill-recommend-hook.py

각 테스트 케이스별로:

1. 실제 매칭된 스킬 목록 추출
2. expectedSkills와 비교
3. mustMatch 스킬이 포함되었는지 확인
4. mustNotMatch 스킬이 제외되었는지 확인

결과를 plugins/hooks/tests/SKILL-RECOMMEND-TEST-REPORT.md 에 다음 형식으로 작성:

## 테스트 결과 요약

- 총 테스트: N개
- 성공: N개 (N%)
- 실패: N개 (N%)
