
# **개발 가이드**: @dev-guidelines.md, @dev-guidelines-success-criteria-template.md

## **OUTPUT-STYLE**

- 언어: 한글로 출력.
- 기술 용어는 영어 가능.
- 온도는 차갑게 출력
- 깔끔하게 군더기 없이 출력

## **서브에이전트 사용 가이드**

- 사용 가능한 서브에이전트 항상 사용 해주세요.
- 적극적으로 서브에이전트를 병렬로 사용 해주세요.

## **지침**

- 증거를 기반한 사실만 말해주세요.
- 추정, 추측은 절대 말하지 마세요. (만약 해야만 하는 상황이라면 **명시**)
- 모르거나, 햇갈리거나, 판단이 안설때는 사용자에게 물어보세요.
