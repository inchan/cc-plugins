# Skills (스킬)

> 사용자 프롬프트에 자동 또는 수동으로 활성화되는 확장 기능

---

## 디렉토리 구조

```
skills/
└── {skill-name}/
    ├── SKILL.md          # 스킬 정의 (필수)
    ├── resources/        # 참고 자료 (선택)
    │   └── *.md
    └── scripts/          # 실행 스크립트 (선택)
        └── *.py|*.sh
```
