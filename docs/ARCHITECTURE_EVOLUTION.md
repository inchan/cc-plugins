## 🎯 개발 원칙

### KISS (Keep It Simple, Stupid)
불필요한 복잡성 제거, 핵심 기능 집중

### YAGNI (You Aren't Gonna Need It)
실제 사용되지 않는 기능 제거

### 80/20 규칙
20%의 핵심 기능이 80%의 가치 제공
